# HDFC-Bank-Dashboard-Using-Python
Python-based financial dashboard project for analyzing HDFC Bank stock performance using KPI cards, price movement visualization, 365-day SMA analysis, monthly returns, volatility measurement, and trading volume insights with interactive charts and financial analytics.

## Project Overview

This project is an interactive financial analytics dashboard developed using Python to analyze the historical stock performance of **HDFC Bank**. The dashboard provides meaningful insights into stock price trends, volatility, trading activity, and technical indicators using visual analytics and statistical calculations.

The project focuses on transforming raw stock market data into an easy-to-understand dashboard for financial analysis and investment insights.

---

# Project Objectives

* Analyze historical HDFC Bank stock performance.
* Visualize stock price trends and technical indicators.
* Measure market volatility and risk.
* Generate monthly return insights.
* Build an interactive and informative stock market dashboard using Python.

---

# Dashboard Features

## Stock Performance Overview (KPI Cards)

Displays key stock performance metrics such as:

* Current Closing Price
* Highest Price
* Lowest Price
* Average Trading Volume
* Percentage Return

---

## Price Movement Analysis

Interactive line chart visualization of:

* Daily closing price trends
* Long-term stock movement patterns
* Historical price behavior

---

## Moving Average Analysis (365-Day SMA)

Technical trend analysis using:

* 365-Day Simple Moving Average (SMA)
* Trend smoothing for long-term investment insights
* Bullish and bearish trend identification

---

## Monthly Return Analysis

Evaluates stock returns on a monthly basis to identify:

* Best-performing months
* Negative return periods
* Return consistency over time

---

## Volatility & Risk Analysis

Risk measurement using rolling volatility calculations:

* Rolling standard deviation
* Market fluctuation analysis
* Risk trend visualization

---

## High vs Low Price Analysis

Area chart comparison of:

* Daily High Prices
* Daily Low Prices
* Price range movement over time

---

## Trading Volume Insights

Volume-based analysis including:

* Highest trading volume days
* Lowest trading volume days
* Trading activity visualization using volume charts
